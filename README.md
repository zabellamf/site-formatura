# Site Formatura
Site feito com Joomla + Docker

# Docker

É utilizado o docker-compose.yml para criar o docker do Joomla configurado.
Para iniciar o projeto é preciso fazer o seguinte comando para executar em background:

```
cd ops

docker-compose up -d 
```
